# ☕ Exercicios-DevMagro

Este repositório contém as minhas resoluções para os exercícios práticos propostos no curso **"Java do básico ao júnior"**, ministrado pelo **@devmagro**

## 🎯 Objetivo

Documentar a evolução do meu aprendizado na linguagem Java, partindo dos conceitos iniciais até a construção de aplicações mais robustas. Este espaço serve como um portfólio prático da minha base como desenvolvedor Backend.

## 🧠 Como os exercícios são resolvidos?

A proposta aqui não é apenas entregar um código que funcione, mas desenvolver uma mentalidade de Engenharia de Software desde a base. Em cada resolução, o foco está em:
* **Legibilidade:** Nomes de variáveis e métodos que deixam clara a intenção do código.
* **Clean Code:** Estruturas simples, diretas e sem complexidade desnecessária.
* **Preparação para o Mercado:** Aplicação gradual de conceitos de Orientação a Objetos e boas práticas (como os princípios iniciais do SOLID) conforme a dificuldade do curso avança.

## 📂 Estrutura do Repositório

Os desafios estão organizados de acordo com a progressão natural do curso. *(Ajuste os nomes das pastas conforme a estrutura real do curso)*:

* 📁 `01-fundamentos-logica/`: Variáveis, tipos de dados e operadores.
* 📁 `02-estruturas-de-controle/`: Condicionais (`if/else`) e loops (`for`, `while`).
* 📁 `03-orientacao-a-objetos/`: Classes, atributos, métodos e encapsulamento.
* 📁 `04-projetos-praticos/`: Exercícios finais e mini-projetos.
